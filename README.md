# 👑 Douglinha Gesso

Website moderno para a empresa **Douglinha Gesso** — serviços profissionais de Drywall, Decoração e Iluminação.

## ✨ Funcionalidades

- 🏠 **Landing page** moderna, limpa e responsiva
- 📸 **Galeria de fotos** com scroll infinito (mais recentes primeiro)
- 🔐 **Painel Admin** para upload de fotos do portfólio
- 💬 **WhatsApp** integrado com botão flutuante
- 📱 **Instagram** com link direto
- 🎨 **Design** branco e dourado, premium

## 🚀 Como usar

1. Abra `index.html` no navegador
2. Para gerenciar fotos: clique ⚙️ → senha `doug2026`
3. Arraste ou selecione fotos → "Publicar Fotos"
